# Movie Search App

#### Vanilla JS, OMDb API를 활용하여 영화 검색 사이트를 개발하였습니다.

### [Demo Web Site](https://kunhee-movie.vercel.app)

```sh
git clone https://github.com/rg5668/vanilla-movie-app.git
```

```sh
npm install
```

```sh
npm run dev
```
